# Proyecto de Análisis de Datos: AquaLimpia S.A.

## Descripción
Este proyecto consiste en el análisis de datos operativos y ambientales de las plantas de tratamiento de **AquaLimpia S.A.** El objetivo principal es identificar causas de incumplimiento en la DBO y optimizar la toma de decisiones.

## Estructura
- `data/`: Dataset original.
- `outputs/`: Reportes (.csv) y dashboard.
- `src/`: Código fuente (`main.py`, funciones).

## Objetivos
* Procesar datos operativos y ambientales.
* Generar reportes automatizados.
* Visualizar el rendimiento mediante un dashboard.
